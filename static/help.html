Frequently Asked Questions

Drivers:

Reserve a parking space by using the navigation bar tabs to go to the "Reserve" page. On this page, use the section on the left to select a floor and a space. Click the "Submit" button to reserve the space.

Change your account information by using the navigation bar tabs to go to the "Account" page. From here, you are able to change any of your account information.

Owners:

As a garage owner, you are able to view your garages by viewing your account information. You can do this by using the navigation bar to go to the "Account" page.

As a garage owner, create a new garage by clicking "Create" in the navigation bar. You will be brought to a page where you can create a new garage.
